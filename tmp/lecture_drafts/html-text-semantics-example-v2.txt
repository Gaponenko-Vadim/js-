## Пример: текстовая структура страницы курса 📄

Разрабатывается условно такой сайт: платформа онлайн‑курсов.
После всех митингов с бизнесом и постановки задачи нужно было решить проблему:
пользователи терялись в каталоге и не понимали, где они находятся.

Ниже — фрагмент готового документа по текстовой семантике страницы курса.

---

### Раздел: Структура описания курса

**Цель:** сделать текст читаемым для пользователя и понятным для поиска.

**Требования:**
1) Использовать один `h1` и последовательные уровни заголовков.  
2) Основной текст оформлен в `p`.  
3) Важные акценты — через `strong` и `em`.  
4) Список преимуществ — через `ul`.  
5) Цитата преподавателя — через `blockquote`.  
6) Дата старта оформлена через `time`.

**Критерии приёмки:**
- Структура читается как логичный рассказ.
- Экранные дикторы видят заголовки и списки.
- Важные акценты имеют смысловую разметку.

**Фрагмент реализации:**
~~~html
<article>
  <h1>Основы HTML</h1>
  <p><strong>HTML</strong> — язык разметки, с которого начинается веб.</p>

  <h2>Кому подойдёт курс</h2>
  <p>Если вы хотите научиться верстать с нуля, этот курс для вас.</p>

  <h2>Что вы получите</h2>
  <ul>
    <li>Понимание структуры страницы</li>
    <li>Навыки работы с текстом</li>
    <li>Практику на реальных задачах</li>
  </ul>

  <h2>Цитата преподавателя</h2>
  <blockquote cite="/about-teacher">
    «Семантика делает текст читаемым и для людей, и для поисковиков»
  </blockquote>

  <p>Старт курса: <time datetime="2026-02-10">10 февраля 2026</time></p>
</article>
~~~
