## Пример: базовый HTML‑каркас 📄

Разрабатывается условно такой сайт: платформа онлайн‑курсов.
После всех митингов с бизнесом и постановки задачи нужно было решить проблему:
пользователи терялись в каталоге и не понимали, где они находятся.

Ниже — фрагмент готового документа по структуре HTML‑документа.

---

### Раздел: Базовый каркас страницы

**Цель:** задать правильную структуру страницы и метаданные для SEO и адаптивности.

**Требования:**
1) Использовать `<!doctype html>` и `lang="ru"`.  
2) В `head` первым стоит `meta charset`.  
3) Добавлены `title` и `meta description`.  
4) Есть `meta viewport` для мобильных.  
5) CSS подключён через `link` в `head`.  
6) Скрипт подключён с `defer`.

**Критерии приёмки:**
- Страница корректно отображается на мобильных.
- Вкладка браузера имеет понятный заголовок.
- Метаданные находятся в `head`.

**Фрагмент реализации:**
~~~html
<!doctype html>
<html lang="ru">
  <head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1" />
    <title>Каталог курсов</title>
    <meta name="description" content="Онлайн‑курсы по веб‑разработке для новичков" />
    <link rel="stylesheet" href="/styles/main.css" />
  </head>
  <body>
    <header>
      <h1>Каталог курсов</h1>
    </header>
    <main>
      <section>
        <h2>Популярные направления</h2>
      </section>
    </main>
    <footer>© 2026 Онлайн‑курсы</footer>

    <script src="/scripts/app.js" defer></script>
  </body>
</html>
~~~
